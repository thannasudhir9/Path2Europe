# Project Documentation: Path to Europe Consulting

## Project Overview
Path to Europe Consulting is a full-stack web application designed to help professionals and fresh graduates transition to careers in Europe. It specializes in IT training (Salesforce), AI-driven job search optimization, and cultural integration.

## Project Structure
- `/src/App.tsx`: Main application component containing all sections (Hero, Services, Gallery, Pricing, Docs, Guidance, Testimonials, About, Contact).
- `/src/index.css`: Global styles using Tailwind CSS.
- `/src/main.tsx`: Entry point.
- `/public/`: Static assets (images, icons).
- `package.json`: Project dependencies and scripts.
- `metadata.json`: App metadata and permissions.

## Key Features
- **Dynamic Navbar:** Smooth scrolling with active section highlighting.
- **Consultation Booking:** Integrated contact form with real-time validation and mailto integration.
- **Pricing Plans:** Tiered pricing for Fresh Graduates and Experienced Professionals.
- **Free Resources (Docs):** Library of guides, resume samples, and official government links.
- **Founder Profiles:** Detailed "About" section featuring Sudhir Kumar Thanna and Sai Lakshmi Harisha.
- **Responsive Design:** Mobile-first approach using Tailwind CSS.
- **Animations:** Smooth transitions using Framer Motion.
